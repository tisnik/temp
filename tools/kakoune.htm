2. Textové editory a další aplikace inspirované ovládáním Vi/Vimu

Ve skutečnosti není textový editor Kakoune v žádném případě jedinou aplikací, která používá ovládání inspirované textovými editory Vi a Vim. Například pro známý textový editor Atom existuje zásuvný modul pojmenovaný atom-vim-mode-plus. Na stránkách serveru mojefedora.cz [1] [2] jste se mohli se seznámit s projektem pojmenovaným Vrapper, jenž přidává klávesové zkratky Vimu do integrovaného vývojového prostředí Eclipse a emulace Vimu je dokonce dostupná o pro Libre Office díky pluginu vibreoffice (ten ovšem podporuje jen ty nejzákladnější příkazy, ze všech zmíněných pluginů je v nejhorším stavu).

Obrázek XX: Integrované vývojové prostředí Eclipse s nainstalovaným pluginem Vrapper.

Jedna z nejdokonalejších emulací Vi a Vimu, s níž se můžeme v praxi setkat, je implementována v pluginu Evil (Mode) pro slavný textový editor Emacs. S možnostmi nabízenými režimem Evil mode jsme se již na stránkách Roota setkali v článku Evil: kombinace editačních příkazů Vimu a síly Emacsu , který byl součástí krátkého seriálu o některých možnostech nabízených jak Emacsem, tak (a to především) Emacs Lispem.

3. Instalace textového editoru Kakoune

Existuje několik možností instalace textového editoru Kakoune. Samozřejmě je možné si stáhnout zdrojové kódy a provést lokální překlad. Kakoune je naprogramován v jazyku C++ (využívá C++17), takže pro překlad vyžaduje jak překladač C++, tak i běžnou sadu nástrojů používaných pro vývoj (make, linker atd.). Kromě toho se Kakoune linkuje i s několika knihovnami, zejména s ncurses (celoobrazovkové textové uživatelské rozhraní). Pokud tedy již máte nainstalovanou vývojovou variantu ncurses a novější překladač jazyka C++, měl by být překlad editoru Kakoune přímočarý:

$ git clone http://github.com/mawww/kakoune.git
$ cd kakoune/src
$ make

popř. následovaný:

$ make install
