1.

V předchozím článku jsme se seznámili s několika možnostmi formátování tabulek (resp. dat organizovaných do formy 2D tabulky) tak, aby se naformátovaná tabulka mohla zobrazit na běžném terminálu s neproporcionálním fontem. Připomeňme si, že popsáno bylo hned několik balíčků, které tvorbu naformátovaných tabulek podporují. Především se jedná o standardní balíček , který dokáže naformátovat prakticky jakýkoli text, v němž jsou jednotlivé buňky výsledné tabulky ukončeny (nikoli ovšem rozděleny) zadaným znakem, jímž je typicky znak Tab (v řetězci je reprezentován symbolem '\t'. Tento standardní balíček má poměrně široké možnosti využití, ovšem nemusí se hodit pro všechny situace.

1..........1..........1
22.........22.........22
333........333........333
4444.......4444.......4444


     1|     1|     1|
    22|    22|    22|
   333|   333|   333|
  4444|  4444|  4444|


package main
 
import (
        "os"
 
        "github.com/olekukonko/tablewriter"
)
 
func main() {
        data := [][]string{
                []string{"A", "The Good", "500"},
                []string{"B", "The Very very Bad Man", "288"},
                []string{"C", "The Ugly", "120"},
                []string{"D", "The Gopher", "800"},
        }
 
        table := tablewriter.NewWriter(os.Stdout)
        table.SetHeader([]string{"Name", "Sign", "Rating"})
 
        for _, v := range data {
                table.Append(v)
        }
        table.Render()
}

Výsledná tabulka vypadá značně odlišně od předchozích tabulek:

| C    | The Ugly              |    120 |
| D    | The Gopher            |    800 |
+------+-----------------------+--------+
|                 SUM          |  1708  |
