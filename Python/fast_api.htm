1.

V dnešním článku si ve stručnosti představíme několik balíčků a technologií, které umožňují snadnou a taktéž rychlou tvorbu webových služeb založených na REST API a vybrané (prakticky libovolné) relační databázi s využitím programovacího jazyka Python. Konkrétně se zaměříme na praktickou ukázku využití frameworku nazvaného FastAPI zkombinovaného s nástrojem SQLAlchemy. Ovšem interně se budou používat i některé další balíčky určené pro ekosystém programovacího jazyka Python, konkrétně balíček Uvicorn a taktéž balíčky Pydantic a Starlette.

V navazujících kapitolách si ukážeme postupnou tvorbu webové služby, která přes čtyři REST API endpointy nabídne uživatelům této služby čtveřici operací CRUD neboli Create, Read, Update a Delete. Tyto operace budou probíhat nad záznamy uloženými v relační databázi. Konkrétně využijeme databázi PostgreSQL, ale relativně snadno bude možné výsledný projekt upravit takovým způsobem, aby se namísto této databáze použila odlišná relační databáze (od SQLite přes MySQL/MariaDB až po „velké“ databáze typu Oracle). REST API endpointy nabízené námi vytvořenou službou budou popsány s využitím známé specifikace OpenAPI (výsledky jsou dostupné přes Swagger a taktéž přes Redoc).

3. Příprava projektu s implementací jednoduché webové služby

V prvním kroku si vytvoříme kostru projektu, který postupně upravíme do formy webové služby. Pro správu projektů využijeme nástroj PDM, který byl popsán v článku PDM: moderní správce balíčků a virtuálních prostředí Pythonu (ovšem stejně dobře můžete použít i klasický pip zkombinovaný s venv nebo virtualenv). from fastapi import FastAPI

class Storage():
    def __init__(self):
        self._items = {}
        self._id = 0

    def create(self, text):
        self._id += 1
        self._items[self._id] = text
        return self._id

    def read(self):
        return self._items

    def update(self, id, text):
        self._items[id] = text

    def delete(self, id):
        del self._items[id]


storage = Storage()

app = FastAPI()

@app.post("/create")
async def create_operation(text: str):
    id = storage.create(text)
    return {"created": id}

@app.get("/")
async def read_operation():
    return {"list": storage.read()}

@app.put("/update/{id}")
async def update_operation(id: int, text: str = ""):
    storage.update(id, text)
    return {"updated": id, "new text": text}

@app.delete("/delete/{id}")
async def delete_operation(id: int):
    storage.delete(id)
    return {"deleted": id}
